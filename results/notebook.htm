The assumptions under which the script works correctly:

·         the command line arguments are valid file paths

·         the input file is in fasta format

·         the input file is a multialignment file

o   the script should not cause an error if at least 1 sequence is present

o   for a meaningful result, at least 3 sequences should be present (perhaps 2, but that is a pairwise alignment instead of a multialignment)

o   all the sequences in the file are of the same length

o   indels are specified by the following character ‘-‘

·         It is not necessary that all the symbols represent amino acids for this script to work. It should work correctly for nucleotide sequences or other types, but for the results to be meaningful, protein sequences should be used. This should be checked by the validation script.

Designing the statistical experiment:

Goal:

                Determine if the noise reduction was worthwhile – whether the tree obtained after removing noisy columns from the multiple sequence alignment resembles the reference tree more closely.

Data and reasoning:

                There are two random variables, the symmetric_difference between the reference tree and the noisy tree and the symmetric_difference between the reference tree and the denoised tree. Let’s call the first variable D_noisy and the latter one D_denoised. What we are interested in finding out is whether the difference between the reference and the denoised tree is less than the difference between the first and the noisy tree, this corresponds to the hypothesis μnoisy > μdenoised, or rather μnoisy - μdenoised > 0. However, since the data of the two variables is paired, ie. for each value of D_noisy we know the corresponding value of D_denoised, the one obtained from the same multiple sequence alignment, we can introduce another random variable D = D_noisy – D_denoised, and treat the differences between corresponding values of D_noisy and D_denoised as observations of D. The hypothesis then becomes μ­­D > 0.

Hypothesis:

                μ­­D > 0

Choosing an appropriate test:

                It is important to determine what type of statistical test should be used. A t-test is appropriate if data is considered to be normally distributed and continuous. Let us assume D is a continuous variable and check how closely is resembles a normal distribution. Let us take a look at D histograms for various data sets (Figure 1):

Figure 1 - Histograms of D for all data sets

                Taking a look at these, the histograms do resemble a normal distribution to an extent. However, I am not convinced yet. Further tests on normality need to be performed.

No difference is observed in the significance of individual tests when using a t-test and a Wilcoxon’s signed-rank test, therefore there is no point in performing further normality tests. Overall, it does not seem worthwhile to perform noise reduction. Out of six data sets, only two report a significant reduction in distance to the reference tree, and one data set shows a significant increase in said distance.
